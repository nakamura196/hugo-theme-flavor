## Tailwind CSS v4

This theme uses Tailwind CSS v4 with Hugo's native `css.TailwindCSS` pipeline.

No PostCSS configuration needed — Hugo handles everything.

## Design Tokens

The theme uses CSS custom properties for theming:

```css
--color-theme: #fafafa;
--color-primary: #1a1a2e;
--color-accent: #4f46e5;
```
